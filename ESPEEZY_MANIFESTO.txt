Alias Information

Unlike some cmdlets that come with widely recognized built-in aliases, Stop-Process is typically used by its full name to maintain clarity. However, you can create your own custom alias if you frequently use this cmdlet.
Example: Set-Alias -Name sp -Value Stop-Process
After setting this custom alias, you can simply type sp -Name "notepad" to stop the Notepad process quickly.








Here are several inline examples illustrating different ways to use Stop-Process:

    To stop all instances of Notepad, type:
    Example: Stop-Process -Name "notepad"

    To terminate a process using its process ID, type:
    Example: Stop-Process -Id 1234

    To stop all processes with names containing “chrome,” type:
    Example: Stop-Process -Name *chrome*

    To remotely stop Notepad on a computer named Server01, type:
    Example: Invoke-Command -ComputerName Server01 -ScriptBlock { Stop-Process -Name "notepad" }

    To create and use a custom alias for Stop-Process, type:
    Example: Set-Alias -Name sp -Value Stop-Process and then use sp -Name “notepad”
